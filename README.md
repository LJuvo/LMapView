# LMapView

## 地图瓦片本地载入示例

#### 已完成功能

- 瓦片载入
- 瓦片动态加载
- 缩放级别控制
- 多缩放级别瓦片载入
- 地图拖拽
- 透明背景
- 自定义瓦片
- 地图图形绘制

## 项目初始化

```
npm install
```

### 项目启动

```
npm run serve
```

### 项目打包

```
npm run build
```

### 项目测试

```
npm run test
```

### Eslint检查

```
npm run lint
```
